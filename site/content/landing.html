Comments for static sites. No JavaScript. No accounts. Just an HTML form. This page is a live demo. Try it out and leave a comment at the bottom of this page.

Source on GitHub.

Inspired by giscus, but different:

  • No client JavaScript — pure HTML forms, no iframe
  • Anonymous — no GitHub account required to comment
  • Cloudflare D1 — comments stored in your own D1 database, not a third-party service. Up to ~25,000 comments per day and ~10 million stored on the Cloudflare free tier
  • Moderation tooling — auto-approve, or review first with approve / reject / spam / ban
  • Optional AI mod — Workers AI blocks spam on submission, legitimate comments go through instantly. ~3,000 classifications per day on the free tier
  • Free — runs on Cloudflare's free tier
  • Works with any static site generator — Hugo, Astro, Eleventy, Jekyll, Next.js and more

How it works

  1. Run npx ziscus init to generate the template and workflow for your static site generator
  2. Deploy the Worker to Cloudflare
  3. The commenter sees their comment instantly. Your site rebuilds automatically
