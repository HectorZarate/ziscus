Comments for static sites. No JavaScript. No accounts. Just an HTML form. This page is a live demo. Try it out and leave a comment at the bottom of this page.

A Cloudflare Worker stores comments in D1, then regenerate your static site to see new comments. Ziscus uses GitHub Actions to regenerate the page automatically. Source on GitHub.

Inspired by giscus, but different:

  • No client JavaScript — pure HTML forms, no iframe
  • Anonymous — no GitHub account required to comment
  • Cloudflare D1 — SQLite at the edge, not GitHub Discussions
  • Optional moderation — auto-approve, or review first with approve / reject / spam / ban
  • Free — runs on Cloudflare's free tier
  • Works with any SSG — Hugo, Astro, Eleventy, Jekyll, Next.js

How it works

  1. Run npx ziscus init to generate the template and workflow for your SSG
  2. Deploy the Worker to Cloudflare
  3. The commenter sees their comment instantly. Your site rebuilds automatically
